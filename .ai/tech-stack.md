# Tech Stack - Promptana

## Core

- App: Next.js + TypeScript, Shadcn, Tailwind
- Auth: Supabase Auth + Postgres
- ORM: Prisma
- Server-Side Calls: Next.js API routes
- Search: Postgres tsvector

## Testing

### Unit Tests
- **Test Runner**: Jest ^29.x
- **React Testing**: React Testing Library ^14.x

### E2E Tests
- **Framework**: Playwright ^1.40
- **Browsers**: Chrome, Firefox, WebKit
